API地址: http://mngame.duapp.com
*号的参数是必须的
返回结果如果出错，返回{err:’错误说明’}，除了记录手机号那如果有异常错误的话，提示再提交试试以外，其它的地方都不需要做额外处理

接口地址 接口描述 返回结果
/init
GET		 用户初始化
/record
POST		 记录得分
*score 分值0-10
/code
GET		 验证码 验证码图片
/lottery
GET		 抽奖
*code验证码		 {status:true,lid:中奖ID} 成功
status:false 失败
/lottery
POST		 记录中奖手机号码
*mobile 手机号
*lid 中奖ID		 status:true 成功
status:false 失败
