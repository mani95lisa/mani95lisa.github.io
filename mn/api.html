API地址: http://mngame.duapp.com
*号的参数是必须的

URI Request [ResultVO][1]
/user/init
GET		 用户初始化，第一次开启应用时调用
*udid 设备号
*rom ROM版本
*product_model 产品型号		 result:[UserVO][2]
/user/login
POST		 用户登录，第一次手机登录后将手机号和验证码记录下来，以后每次启动应用都登录一次
*udid 设备号
*mobile手机号
*code 验证码		 result:[UserVO][2]
/user/code
POST		 获取验证码，60秒内只能获取一次
*mobile 手机号
*udid 设备号		 status:true 成功
status:false 失败
