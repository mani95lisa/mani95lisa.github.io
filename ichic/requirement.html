流程图

功能列表

App

功能名称 描述 备注
登录 通过微信、微博、QQ账号等第三方平台或邮箱密码进行登录 1.账号密码非空、邮箱格式校验
2.校验或登录失败显示相应提示
3.通过第三方平台登录使用其默认昵称作为用户名，是否允许相同昵称的用户存在？
注册 通过邮箱、密码进行注册 1.是否需要邮箱验证?
2.是否需要发送欢迎邮件?
3.邮箱及密码不超过50字符
找回密码 发送注册邮箱验证码，用以用户重置密码 1.第三方平台用户没有密码，无法找回，只能通过第三方账号进行登录
2.验证码为4位数字，校验成功即可进行登录

管理系统

功能名称 描述 备注
管理员管理 系统管理员管理
用户管理 所有用户统一管理：
1.设置用户身份，分为普通用户和认证用户，可以录入认证信息
2.查看用户信息，包括昵称、邮箱、发布的搭配数、关注人数、粉丝人数、登录次数、最近一次登录时间、账号创建时间
3.禁用及启用用户		 1.支持按昵称检索
2.支持分页
3.支持按搭配数、关注数、粉丝数、登录次数排序
邮件模板 欢迎邮件模板设置编辑 如果需要发送欢迎邮件的话

数据模型

User

属性 描述 备注
id 唯一ID
email 邮箱 不超过50个字符
password 密码 MD5加密
nickname 昵称 不超过10个字符
truename 真实姓名 备用
mobile 手机 备用
following 关注数
followers 粉丝数
styles 搭配数
come_from 用户来源 用以区分应用渠道，比如Android版不同市场等
loginAttempts 登录尝试次数 超过5次密码错误禁止登录2小时
lockUntil 禁止登录截止时间
last_signed_in_time 最近一次登录时间
signed_in_times 登录次数
access_token 第三方平台授权秘钥
portrait 头像
platform 第三方平台标识
usid 第三方平台用户ID
disabled 是否已禁用
birthday 生日
address 地址
gender 性别
weight 体重
height 身高
verify_code 验证码
updated_at 更新时间
created_at 创建时间
